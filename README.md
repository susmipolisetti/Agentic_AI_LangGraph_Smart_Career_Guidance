# 🚀 Agentic AI with LangGraph - Smart Career Guidance Agent

## 📌 Project Overview

This project is an AI-powered Smart Career Guidance Agent developed using Python and LangGraph concepts. It helps students identify skill gaps, recommends suitable career paths, suggests learning roadmaps, and provides job recommendations based on their skills.

---

## ✨ Features

- Resume Analysis
- Skill Gap Detection
- Career Recommendation
- Learning Roadmap Generation
- Job Matching
- Report Generation

---

## 🛠 Technologies Used

- Python
- LangGraph
- LangChain
- OpenAI / Gemini API
- FAISS Vector Database

---

## 📂 Project Structure

```
app.py
agents.py
workflow.py
resume_parser.py
job_matcher.py
report_generator.py
career_dataset.csv
requirements.txt
```

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```
