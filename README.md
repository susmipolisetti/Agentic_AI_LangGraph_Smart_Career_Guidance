# 🚀 Agentic AI with LangGraph – Smart Career Guidance Agent

## 📖 Overview

The Smart Career Guidance Agent is an AI-powered application developed using Python and LangGraph concepts. It analyzes resumes, detects skill gaps, recommends career paths, generates learning roadmaps, and suggests relevant job opportunities.

---

## ✨ Features

- Resume Analysis
- Skill Gap Detection
- Career Recommendation
- Learning Roadmap Generation
- Job Matching
- Career Guidance Report

---

## 🛠 Technologies Used

- Python
- LangGraph
- LangChain
- OpenAI API / Gemini API
- FAISS Vector Database
- GitHub
- VS Code
