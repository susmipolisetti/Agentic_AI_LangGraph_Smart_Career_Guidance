# Agentic AI with LangGraph – Smart Career Guidance Agent

## Project Overview
This project is developed using LangGraph and Large Language Models (LLMs) to provide intelligent career guidance. The system analyzes resumes, identifies skill gaps, recommends suitable career paths, generates personalized learning roadmaps, and suggests relevant job opportunities.

## Technologies Used
- Python
- LangGraph
- LangChain
- OpenAI API / Gemini API
- FAISS Vector Database
- VS Code
- GitHub

## Features
- Resume Analysis
- Skill Gap Detection
- Career Recommendation
- Learning Roadmap Generation
- Job Matching
- Career Guidance Report
